urllib概览

urllib.request.urlopen

  • url
    • 你想爬的网址
    • Get请求
    • 接收网址
    • Requests对象 可以接受
  • data
    • POST请求
    • data传递的是表单
  • timeout
    • 超时时间
    • 在规定的时间内没有返回就会报错

httpbin

  • http://httpbin.org/get
    • 得到get请求
  • post
    • 传递表单
  • IP
    • 返回本地IP地址

  • 注意请求头 是 Python
